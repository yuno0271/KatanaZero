# KatanaZero

## 프로젝트 소개
액션게임 카타나제로의 게임성과 액션성을 유니티에서 구현해보고자 제작했습니다

## 팀 프로젝트 개요
- 제작기간 : 3주
- 개발도구 : Unity
- 개발언어 : C#
- 플랫폼 : PC

### 참여 부분
Player가 행하는 모든 액션과 플레이어와 관련한 모든 스크립트,
UI와 플레이어간의 스크립트 연결, 총알 이펙트와 모든 사운드
